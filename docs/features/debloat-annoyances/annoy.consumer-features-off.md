# Disable Windows consumer features · 停用 Windows 消費者功能

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `annoy.consumer-features-off` |
| **Module · 模組** | Debloat & Annoyances · 去煩擾 |
| **Type · 種類** | Toggle |
| **Administrator · 管理員** | Yes · 需要 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | SignOut |

## English
Stop Windows from auto-installing promoted apps and showing consumer-oriented suggestions.

## 粵語
唔再俾 Windows 自動裝推廣 App 同顯示消費者導向嘅建議呀。
