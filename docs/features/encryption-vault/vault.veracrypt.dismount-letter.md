## English
Guided dismount — pick a currently-mounted drive letter and optionally force it even if files are open; the same engine dismount runs at the end.

## 粵語
引導式卸載 — 揀一個已掛載嘅磁碟機代號，可選即使有檔案開住都強制卸載；最後行返同一個引擎卸載流程。
