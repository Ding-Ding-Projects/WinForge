# Disable wallpaper JPEG compression · 停用桌布 JPEG 壓縮

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `appearance.wallpaper-quality` |
| **Module · 模組** | Appearance & Personalisation · 外觀與個人化 |
| **Type · 種類** | Toggle |
| **Administrator · 管理員** | No · 唔使 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | None |

## English
Set wallpaper import quality to 100 so Windows stops recompressing JPG wallpapers (re-applies the current wallpaper).

## 粵語
將桌布匯入品質設為 100，唔再壓縮 JPG 桌布（會即時重新套用目前桌布）。
