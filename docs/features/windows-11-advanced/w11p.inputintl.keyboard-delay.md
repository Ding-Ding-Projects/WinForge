# Keyboard repeat delay · 鍵盤重複延遲

| Field · 欄位 | Value · 值 |
|---|---|
| **ID** | `w11p.inputintl.keyboard-delay` |
| **Module · 模組** | Windows 11 Advanced · Windows 11 進階 |
| **Type · 種類** | Slider |
| **Administrator · 管理員** | No · 唔使 |
| **Destructive · 具破壞性** | No · 唔係 |
| **Restart · 重啟** | None |

## English
How long to hold a key before it repeats (0 = shortest, 3 = longest).

## 粵語
撳住一個鍵幾耐先開始重複（0 = 最短，3 = 最長）。
