# Files & Disks · 檔案與磁碟

Tools for compressing, searching, previewing and recovering files, plus drive management, health checks and benchmarks. · 用嚟壓縮、搜尋、預覽同救援檔案嘅工具，仲有磁碟管理、健康檢查同速度測試。

## Archives · 壓縮檔

Compress, extract and test ZIP/7z/RAR/TAR archives. · 壓縮、解壓同測試 ZIP／7z／RAR／TAR 壓縮檔。

Open in-app: `WinForge.exe --page archives`

![Archives](images/screenshot-archives.png)

## Batch Rename · 批次改名

Bulk rename files with regex and sequence patterns. · 用正規表示式同序號樣式批次改名。

Open in-app: `WinForge.exe --page rename`

![Batch Rename](images/screenshot-rename.png)

## Bulk File Ops · 批次檔案操作

Mass move, copy, delete and set attributes on files. · 批量移動、複製、刪除同設定檔案屬性。

Open in-app: `WinForge.exe --page bulkops`

![Bulk File Ops](images/screenshot-bulkops.png)

## New+ · 範本新增

Create files and folders from templates in the New menu. · 由範本喺「新增」選單建立檔案同資料夾。

Open in-app: `WinForge.exe --page newplus`

![New+](images/screenshot-newplus.png)

## Duplicate Finder · 重複檔案搜尋

Open in-app: `WinForge.exe --page duplicates`

![Duplicate Finder](images/screenshot-duplicates.png)

## Instant File Search · 即時檔案搜尋

Open in-app: `WinForge.exe --page everything`
