# Package Manager · 套件管理

![Package Manager](https://raw.githubusercontent.com/codingmachineedge/WinForge/main/docs/screenshot-packages.png)

**EN —** One UniGetUI-style front-end over eight package managers — winget, Scoop, Chocolatey, pip, npm, .NET tools, PowerShell Gallery and Cargo. Discover / Updates / Installed / Bundles (export-import) / Setup tabs, per-manager filtering, batch update and one-click engine bootstrap.

**粵語 —** 一個 UniGetUI 式介面統管八個套件管理器 — winget、Scoop、Chocolatey、pip、npm、.NET 工具、PowerShell Gallery 同 Cargo。搜尋／更新／已安裝／清單（匯出匯入）／設定分頁、逐個引擎篩選、批次更新同一鍵安裝引擎。
