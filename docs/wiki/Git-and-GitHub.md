# Git & GitHub · Git 與 GitHub

![Git & GitHub](https://raw.githubusercontent.com/codingmachineedge/WinForge/main/docs/screenshot-git.png)

**EN —** A multi-repository workspace: add, scan or clone repos, review repository health, manage remotes, stashes and tags, then stage / commit / branch / sync. The chunked uploader splits a large push into N-MB-per-commit pieces and pushes them one at a time. A free-form git & gh command runner sits alongside a 553-operation library covering the complete git CLI and everything GitHub exposes through gh / gh api.

**粵語 —** 一個多儲存庫工作區：加、掃描或複製 repo，睇儲存庫健康狀態，管理 remotes、stashes 同 tags，再暫存／提交／分支／同步。分批上載會將大型 push 切成每個 commit N-MB，逐個推上去。仲有一個自由格式嘅 git 同 gh 指令執行器，配埋 553 個操作嘅資料庫，涵蓋成個 git CLI 同 GitHub 透過 gh／gh api 提供嘅一切。
