# Apps, Git & Packages · 應用程式、Git 與套件

Tools for source control, package managers, app removal, Android device workflows, torrents and communications. · 集合咗源碼管理、套件管理員、移除應用程式、Android 裝置流程、種子下載同通訊嘅工具。

## Git & GitHub · Git 與 GitHub

Multi-repo workbench for git and gh operations with a chunked uploader. · 多儲存庫工作台，操作 git 同 gh，附分塊上傳器。

Open in-app: `WinForge.exe --page git`

![Git & GitHub](https://raw.githubusercontent.com/codingmachineedge/WinForge/main/docs/screenshot-git.png)

## Package Manager · 套件管理

One front-end over winget, scoop, choco, pip, npm and more. · 統一前端操作 winget、scoop、choco、pip、npm 等。

Open in-app: `WinForge.exe --page packages`

![Package Manager](https://raw.githubusercontent.com/codingmachineedge/WinForge/main/docs/screenshot-packages.png)

## App Uninstaller · 應用程式解除安裝
