# Connections (SSH / network) · 連線

![Connections (SSH / network)](https://raw.githubusercontent.com/codingmachineedge/WinForge/main/docs/screenshot-connections.png)

**EN —** Every live TCP/UDP socket and the app that owns it — protocol, local and remote address, state and owning process/PID. Filter by process, address or port, then drop a single connection or end the process. No Resource Monitor needed. (This is the network-connections surface the suite groups under SSH / Connections; it is not a dedicated SSH client.)

**粵語 —** 每一條即時 TCP/UDP 連線同擁有佢嘅程式 — 協定、本機同遠端位址、狀態同擁有嘅程序／PID。可以用程序、位址或連接埠篩選，再單獨切斷一條連線或結束程序。唔使開資源監視器。（呢個就係套件歸入 SSH／連線嘅網絡連線介面，唔係一個專門嘅 SSH 客戶端。）
