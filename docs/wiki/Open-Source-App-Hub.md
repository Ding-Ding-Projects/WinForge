# Native OSS Clones · 開源原生分頁

**EN —** This page documents the native-only rule for open-source-inspired additions: useful upstream app ideas are remade as WinForge C# tabs, not shipped as installer-only shortcuts. A native tab may use managed libraries, Windows APIs, or a focused C# reimplementation, but the user workflow stays inside WinForge.

**粵語 —** 呢頁記錄受開源 app 啟發嘅新增功能規則：有用嘅上游 app 想法要重製成 WinForge C# 分頁，而唔係只提供安裝捷徑。原生分頁可以用受控程式庫、Windows API 或集中式 C# 重新實作，但使用流程要留喺 WinForge 入面。

Open in-app: `WinForge.exe --page ossapps`

## Native Clone Index · 原生複製索引

- Developer & data · 開發與數據: API Client, Diff & Merge, Diagram Editor, .NET Decompiler, SQLite Browser.
- Documents & knowledge · 文件與知識: Feed Reader, Flashcards.
- Media & documents · 媒體與文件: PDF Toolkit, Audio Tagger, Image Editor, Text Extractor.
- Security & privacy · 安全與私隱: KeePass Vault.
- Network & transfer · 網絡與傳輸: Native Torrent.
- Virtualization & containers · 虛擬化與容器: Docker.
- System utilities · 系統工具: Process Explorer, Disk Health, Disk Benchmark, Everything Search.

## New Native Clone · 新增原生複製

**Feed Reader · RSS 閱讀器** is the newest native clone. It is inspired by QuiteRSS and Fluent Reader, but implemented inside WinForge with C# `HttpClient`, XML parsing, local JSON feed storage, article lists and in-app summaries.

**RSS 閱讀器**係最新原生複製。佢受 QuiteRSS 同 Fluent Reader 啟發，但喺 WinForge 入面以 C# `HttpClient`、XML 解析、本機 JSON feed 儲存、文章清單同 app 內摘要實作。
