# Native C++ Port Moved · 原生 C++ 移植版已搬遷

The experimental C++20/C++/WinRT port of WinForge is now an independent project:

WinForge 嘅實驗性 C++20/C++/WinRT 移植版而家係獨立 project：

## [codingmachineedge/WinForge-Native](https://github.com/codingmachineedge/WinForge-Native)

Its source, standard-C++ core, C++/WinRT shell, tests, parity ledger, installer, build/release automation, roadmap, and detailed feature evidence are maintained there. This repository remains the canonical .NET 11 / WinUI 3 application and managed release line.

原生版嘅 source、標準 C++ core、C++/WinRT shell、tests、parity ledger、installer、建置／發佈自動化、路線圖同詳細功能證據，都會喺新 repository 維護。呢個 repository 繼續係正式 .NET 11／WinUI 3 app 同 managed release 線。

Historical native commits, tags, and releases remain in this repository's history for provenance. New native development and releases belong only in WinForge-Native.

歷史原生 commit、tag 同 release 會留喺呢個 repository 歷史作來源記錄；新原生開發同 release 只可以喺 WinForge-Native。
