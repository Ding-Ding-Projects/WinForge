# Cloudflare & Tunnel · Cloudflare 與 Tunnel

![Cloudflare & Tunnel](https://raw.githubusercontent.com/codingmachineedge/WinForge/main/docs/screenshot-cloudflare.png)

**EN —** Drive cloudflared straight from WinForge: named tunnels, free quick (trycloudflare) tunnels, DNS routing, Cloudflare Access, DNS-over-HTTPS and WARP. Operations use editable placeholders (tunnel name, hostname, local service URL); long-running ones open in a terminal. If cloudflared is missing, a one-click winget install bootstraps it.

**粵語 —** 喺 WinForge 直接驅動 cloudflared：具名 tunnel、免費快速（trycloudflare）tunnel、DNS 路由、Cloudflare Access、DNS-over-HTTPS 同 WARP。操作用可編輯嘅佔位符（tunnel 名、主機名、本機服務 URL）；長時間運行嘅會喺終端機開出嚟。冇 cloudflared 嘅話，一鍵 winget 安裝幫你裝好。
