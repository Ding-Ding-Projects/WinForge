# System Monitor · 系統監察

![System Monitor](https://raw.githubusercontent.com/codingmachineedge/WinForge/main/docs/screenshot-monitor.png)

**EN —** A live system monitor: real-time CPU, RAM and network read-outs, plus a process list where you can set priority and CPU affinity — a focused alternative to Task Manager.

**粵語 —** 一個即時系統監察器：實時 CPU、RAM 同網絡讀數，加埋一個程序清單，可以設定優先順序同 CPU 親和性 — 一個專注嘅工作管理員替代品。
