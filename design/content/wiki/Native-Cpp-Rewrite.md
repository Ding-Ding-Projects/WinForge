# Native C++ Port Moved · 原生 C++ 移植版已搬遷

The experimental C++20/C++/WinRT port now lives at [codingmachineedge/WinForge-Native](https://github.com/codingmachineedge/WinForge-Native), with independent source, tests, parity evidence, automation, documentation, and releases.

實驗性 C++20/C++/WinRT 移植版而家喺 [codingmachineedge/WinForge-Native](https://github.com/codingmachineedge/WinForge-Native)，獨立保存 source、tests、parity 證據、自動化、文件同 release。

This site documents the canonical .NET 11 / WinUI 3 WinForge application.

Legacy C++/WinRT checkout refs are retained in the standalone native repository before their old worktrees are retired; this does not put rewrite source back into managed main. Date, Duration, and Loan remain pushed native WIP, not native-main integrations.

舊 C++/WinRT checkout ref 會喺退役 worktree 前先保留喺獨立 native repository；唔會將 rewrite source 放返入 managed main。Date、Duration、Loan 仍然係已 push native WIP，唔係 native-main 整合。
