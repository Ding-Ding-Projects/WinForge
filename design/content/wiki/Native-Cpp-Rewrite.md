It provides safe read-only Discover/Updates/Installed queries, Details and Sources probes, update rules, native preferences, and a bounded v3 Bundle metadata workspace across nine views and eleven Windows manager adapters. · 原生 Package Manager 仍然進行中，唔係完整 UniGetUI 複製。佢喺九個 view 同十一個 Windows manager adapter 提供安全只讀 Discover／Updates／Installed 查詢、Details 同 Sources 探測、更新規則、原生偏好同有界 v3 Bundle metadata 工作區。

An individual cached Install/Update/Uninstall row can become an `AwaitingConsent` record with a redacted reviewed argv. A separate Confirm action is needed before serial normal-integrity execution. ## Native regex search and builder · 原生正規搜尋同建立器

`RegexSearchSurface.h` registers Shell catalog, All Apps, and cached Package Discover as the complete current native search set. Each uses bounded non-JIT PCRE2-16. The full four-step `module.regextester` builder handles flags, escaped literal/route/package/version recipes, safe composition, groups/alternation/word-boundary/lookaround assertions/quantifiers, capture preview, and valid-only application to a chosen native target. Package regex is cache-only, never enters argv/HTTPS, disables remote Search, forces Discover when applied through the builder, and its `NativePackageQueryAudit` changes only for a real package query. PCRE2 attribution ships in `THIRD-PARTY-NOTICES.txt`, the portable ZIP, and the installer. · `RegexSearchSurface.h` 登記咗 Shell 目錄、所有 app 同已快取 Package Discover 做而家完整原生搜尋集合；每個都用有界、非 JIT PCRE2-16。完整四步 `module.regextester` 建立器處理旗標、已 escape literal／路線／套件／版本 recipe、安全組合、group／alternation／word-boundary／lookaround assertion／quantifier、capture 預覽，同只會套用有效模式去已揀原生目標。Package regex 只限快取、唔會入 argv／HTTPS、會停用遠端 Search、由 builder 套用時會強制 Discover，而且 `NativePackageQueryAudit` 只會喺真正套件查詢時改。PCRE2 歸屬會放喺 `THIRD-PARTY-NOTICES.txt`、可攜 ZIP 同 installer。

## Native UUID v7 · 原生 UUID v7

`module.uuidv7` is a native C++ RFC 9562 generator/decoder, available through `uuidv7` and its canonical route. It bulk-generates 1–1000 canonical values from the 48-bit Unix-millisecond field and cryptographic randomness; optional monotonic mode handles same-millisecond creation, backwards clocks, and `rand_a` overflow. The local decoder accepts canonical, compact, braced, and `urn:uuid:` text; reports version/variant and v7/best-effort-v1 UTC/local time; canonicalizes lowercase output; and clears stale results after invalid input. Explicit Copy is the only clipboard effect, and state survives language changes. The route is `in-progress` only because its fresh visual capture is blocked. · `module.uuidv7` 係原生 C++ RFC 9562 產生／解碼器，可以用 `uuidv7` 同本體 route 開啟。佢由 48-bit Unix millisecond 欄位加密碼學隨機值批量產生 1–1000 條標準值；可選 monotonic 模式處理同一毫秒建立、時鐘倒退同 `rand_a` overflow。本機 decoder 接受標準、緊湊、花括號同 `urn:uuid:` 文字；報告 version／variant 同 v7／best-effort-v1 UTC／本機時間、標準化小楷輸出，並喺無效輸入後清除舊結果。明確 Copy 係唯一剪貼簿效果，而狀態會喺轉語言後保留。route 只因最新視覺擷取受阻先係 `in-progress`。
